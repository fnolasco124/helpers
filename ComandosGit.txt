Instalacion:
-----------
	sudo apt-get install git-core


Clonacion de un repositorio
---------------------------
	git clone git://github.com/fnolasco124/helpers.git

Actualizacion de la informacion del repositorio
-----------------------------------------------
	git pull
	
Inicializar el repositorio
--------------------------
	git init
	git remote add origin git://github.com/fnolasco124/helpers
	git config --global user.name "nombre"
	git config --global user.email "email"
